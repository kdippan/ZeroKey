What is the Web Crypto API?

The Web Crypto API is a powerful native interface built directly into modern web browsers like Chrome, Firefox, Safari, and Edge. It provides developers with a set of low-level cryptographic primitives that allow JavaScript applications to perform complex security operations—such as hashing (SHA-256), signature generation, key derivation (PBKDF2), and authenticated encryption (AES-GCM)—without relying on heavy, potentially vulnerable third-party node modules or backend servers.

Why is Client-Side Cryptography Important?

Traditional web applications handle security by sending plaintext data over TLS to a server, where the backend encrypts it before storing it in a database. This architecture inherently means the server owner has access to your unencrypted data prior to encryption. By utilizing the SubtleCrypto interface, zero-knowledge platforms like ZeroKey can scramble your secrets locally on your device. The server only receives a mathematically secure, unreadable blob of ciphertext, ensuring true privacy and end-to-end encryption.

Why check for window.crypto.subtle?

The window.crypto.subtle object is where the actual mathematical heavy lifting occurs. For security reasons, modern browsers restrict access to this interface strictly to secure contexts. This means if a website is loaded over an insecure HTTP connection, the browser will intentionally block access to SubtleCrypto to prevent man-in-the-middle (MITM) attacks from intercepting or manipulating the cryptographic operations.
