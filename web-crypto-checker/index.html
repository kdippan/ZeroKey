What is the Web Crypto API?

The Web Crypto API is a native interface built directly into modern web browsers. It provides a set of low-level cryptographic primitives that allow JavaScript applications to perform complex security operations—such as hashing, signature generation, and encryption—without relying on third-party libraries or backend servers.

Why is Client-Side Cryptography Important?

Traditional web applications send plaintext data to a server, where the server encrypts it before storing it. This means the server owner has access to your unencrypted data. By utilizing the Web Crypto API, zero-knowledge platforms like ZeroKey can scramble your data locally on your device. The server only ever receives a mathematically secure, unreadable blob of ciphertext.

Why check for SubtleCrypto?

The window.crypto.subtle interface is where the actual heavy lifting happens. Browsers restrict access to this interface strictly to secure contexts (HTTPS). If a website is loaded over an insecure HTTP connection, the browser will block access to SubtleCrypto to prevent man-in-the-middle attacks from intercepting the cryptographic operations.
