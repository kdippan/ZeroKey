The Cryptographic Engine

ZeroKey utilizes the native Web Crypto API built directly into your browser. When you encrypt a payload, we use AES-256-GCM (Advanced Encryption Standard with Galois/Counter Mode).

  • No Server Trust: We never receive your plaintext or your decryption keys.
  • Tamper-Proof: GCM mode ensures authenticated encryption. If a single bit is altered in transit, decryption fails safely.
  • Ephemeral: Data is violently purged from PostgreSQL the moment it is read.
