What is AES-256-GCM?

The Advanced Encryption Standard (AES) is a symmetric block cipher established by the U.S. National Institute of Standards and Technology (NIST). The "256" refers to the 256-bit key size, which provides military-grade security that is currently considered computationally impenetrable against brute-force attacks.

GCM (Galois/Counter Mode) is a specific mode of operation for AES. Unlike older modes like CBC, GCM provides Authenticated Encryption. This means it not only encrypts the data to ensure confidentiality, but it also generates an authentication tag to guarantee data integrity. If a malicious actor alters even a single bit of the ciphertext, the decryption process will fail entirely, preventing tampering attacks.

How this encryption tool works

This tool utilizes the native window.crypto.subtle interface to perform all cryptography directly inside your browser. No data is sent to an external server.

  1. Key Derivation: Your text password is mathematically stretched using PBKDF2 (with a random salt and 100,000 iterations) to derive a cryptographically strong 256-bit AES key.
  2. Initialization Vector: A random 12-byte Initialization Vector (IV) is generated. This ensures that encrypting the same message twice produces completely different ciphertexts.
  3. Encryption: The plaintext is encrypted using AES-GCM, producing the ciphertext and the authentication tag.
  4. Packaging: The Salt, IV, and Ciphertext are concatenated together and encoded into a single Base64 string so you can easily copy and paste it.
